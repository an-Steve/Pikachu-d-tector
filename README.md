## ⚡ Pikachu Detector - CNN Image Classifier
### Réalisé par ANTON NELCON Steve

<img width="1828" height="867" alt="image" src="https://github.com/user-attachments/assets/00c40de4-5cd2-46f6-8324-70e9a4fdb621" />

Pikachu Detector est un classificateur d'images basé sur un réseau de neurones convolutif (CNN) qui reconnaît le célèbre Pokémon Pikachu directement dans votre navigateur. Aucun serveur n'est requis - toute l'analyse se fait côté client grâce à TensorFlow.js !
Le projet a été conçu comme une démonstration éducative de machine learning accessible, inspiré par le Mario Detector.

# Points forts

→  Détection en temps réel - Analyse en moins d'une seconde
→  Interface moderne - Design responsive aux couleurs de Pikachu
→  100% local - Aucune donnée n'est envoyée à un serveur
→  Analyse détaillée - Scores de confiance et caractéristiques détectées
→  Mode debug - Affichage des valeurs exactes pour le diagnostic
→  Responsive - Fonctionne sur desktop, tablette et mobile

Fonctionnalités
✅ Détection précise de Pikachu avec ~90% de précision
✅ Analyse CNN multi-critères (couleur, symétrie, forme)
✅ Interface glisser-déposer intuitive
✅ Prévisualisation de l'image en temps réel
✅ Scores détaillés par catégorie
✅ Mode debug pour diagnostiquer les résultats
✅ Animations fluides et design moderne
✅ Compatible avec tous les navigateurs modernes
✅ Fonctionne hors ligne après le premier chargement
